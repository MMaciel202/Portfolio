Como Funciona

1. A demo consulta a disponibilidade no calendário Appointments.
2. A API valida conflitos no Outlook e no Supabase.
3. O booking cria o evento, salva o registro e gera um token seguro.
4. O cancelamento remove o evento e libera o horário.
